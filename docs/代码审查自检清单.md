# 献血管理系统代码审查自检清单

## 1. 项目结构

- [x] Maven 项目可正常识别和编译。
- [x] Java 主类位于 `com.blooddonation.Main`。
- [x] DAO、服务、UI、配置、异常和 DTO 职责分开。
- [x] MySQL DAO 位于 `dao/mysql`。
- [x] MongoDB DAO 位于 `dao/mongo`。
- [x] SQL 和 MongoDB 脚本位于 `src/main/resources/sql`。
- [x] 测试位于 `src/test/java`。
- [x] 没有引入与当前功能无关的框架或依赖。

## 2. MySQL 数据访问

- [x] 核心业务表包含 `users`、`categories`、`items`、`orders`、`profiles`。
- [x] 每张表都有主键。
- [x] 表之间的关联使用外键约束。
- [x] 用户名、邮箱、用户档案和证件号有唯一约束。
- [x] 数量和状态字段有检查约束。
- [x] 常用关联字段、状态字段和时间字段已建立索引。
- [x] DAO 查询使用 `PreparedStatement`。
- [x] 用户输入不直接拼接到 SQL。
- [x] JDBC 连接、语句和结果集使用 try-with-resources 关闭。
- [x] MySQL 连接由 HikariCP 管理。
- [x] 查询结果在 `BaseDAO` 中统一转换。

## 3. MySQL 高级对象

- [x] 用户档案、库存汇总、待审批申请和月度用血汇总视图已提供。
- [x] 月度报表存储过程已提供并由 Java 调用。
- [x] 分类报表和用户申请报表存储过程已提供。
- [x] 库存数量和状态触发器已提供。
- [x] 申请数量、状态和审批条件触发器已提供。
- [x] 触发器只负责兜底校验，核心申请审批仍由 Java 事务控制。

## 4. MongoDB 数据访问

- [x] `action_logs` 保存用户行为。
- [x] `comments` 保存评论、评分和标签。
- [x] `item_details` 保存库存详情、图片列表和扩展属性。
- [x] `system_logs` 保存登录、警告、错误和关键操作日志。
- [x] MySQL 的业务 ID 作为 MongoDB 文档关联字段。
- [x] 日志、评论和详情的高频查询字段有索引。
- [x] `item_details.item_id` 有唯一索引。
- [x] 热门批次、评分和审计报表使用 MongoDB 聚合管道。
- [x] 日志和评论查询包含数量上限。

## 5. 事务和数据一致性

- [x] 创建申请时显式关闭自动提交。
- [x] 事务成功后执行 `commit()`。
- [x] 事务异常时执行 `rollback()`。
- [x] 事务结束后恢复自动提交。
- [x] 审批申请时使用 `SELECT ... FOR UPDATE` 锁定记录。
- [x] 审批前检查申请状态、库存状态和库存数量。
- [x] 审批通过时在同一事务中更新申请和扣减库存。
- [x] 删除已完成申请时在同一事务中恢复库存。
- [x] 已完成或已取消的申请不能再修改状态。
- [x] MySQL 为核心业务数据的权威数据源。
- [ ] MySQL 和 MongoDB 写入失败后没有自动补偿队列。

说明：课程项目不引入分布式事务。MongoDB 详情和日志属于扩展数据，核心库存和申请状态仍由 MySQL 保证。

## 6. 身份认证和权限

- [x] 用户密码使用 BCrypt 哈希保存。
- [x] 登录时使用 BCrypt 校验密码。
- [x] 账号停用后不能登录。
- [x] 登录成功会写入系统日志。
- [x] 登录失败会写入警告日志。
- [x] 普通用户不显示用户管理、分类管理和系统日志入口。
- [x] 普通用户只能维护自己的档案。
- [x] 普通用户只能修改或删除自己的待审批申请。
- [x] 普通用户只能删除自己的评论。
- [x] 只有超级管理员可修改其他用户的角色。
- [x] 禁止删除超级管理员。
- [x] 禁止删除当前登录账号。
- [ ] 注册密码未设置最小长度和复杂度要求。
- [ ] 登录失败未设置限流或临时锁定。

## 7. 输入校验和错误处理

- [x] 登录用户名和密码不能为空。
- [x] 注册邮箱执行格式校验。
- [x] 注册手机号执行 11 位数字校验。
- [x] 档案邮箱、手机号、姓名和证件号执行校验。
- [x] 库存和申请数量必须大于 0。
- [x] 申请数量不得超过可用库存。
- [x] 评论内容不能为空。
- [x] 评分必须为 1 至 5。
- [x] 分类父子关系不能形成自引用或循环。
- [x] 数据库连接异常会转换为项目统一异常或界面提示。
- [x] 界面错误提示使用可理解的中文信息。
